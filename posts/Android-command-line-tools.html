Android命令行工具

android

下面介绍命令行下使用android脚本。

场景：命令行下安装android build tools、sdk、API。

获取所有可用的包:

$ android list sdk -a -u
$ android list sdk -a -u --proxy-host proxy-host --proxy-port proxy-port  # 设置代理

安装指定的包：(num1, num2是上面命令返回的序号)

$ android update sdk -u -a -t <num1,num2>
